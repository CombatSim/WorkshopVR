Package Description
This package contains two pieces of industrial metalworking equipment a manual Mill and a manual Lathe.  This equipment survived a war or an apocalyptic event.  Both pieces are highly distressed by the time and elements; however, they are still in working order.  You can see peeling paint and rust.
The Mill is a 3D Model with 1500 polys and 2048X2048 Diffuse Texture Map and a Normal Map.
The Lathe is 3D Model with 2500 polys and 2048X2048 Diffuse Texture Map and a Normal Map.
This set was designed for possible use in an upcoming RPG called “Wasteland 2” by inXile entertainment.  It is showcased in the Seed Scene provided by inXile entertainment.
